## 我的hexo博客源码

### 仓库说明

##### 使用说明

```bash
hexo clean #清除了你之前生成的东西，也可以不加。
hexo generate #生成静态文章，可以用 hexo g缩写
hexo deploy #部署文章，可以用hexo d缩写
hexo server #启动hexo服务，可以用hexo s缩写。浏览器输入localhost:4000就可以看到你生成的博客
```

##### npx配置

由于我的华硕笔记本没有配置npm的环境变量，所以在使用上述命令时，需要在其前加上`npx`方可。
